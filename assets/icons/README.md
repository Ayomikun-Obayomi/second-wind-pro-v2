# Icon assets

SVG mirrors of icons registered in `js/icons.js`. Sourced from **Untitled UI Line icons** in Figma.

Add new files here when expanding the set: `assets/icons/<token>.svg`

Each file should:
- Use `viewBox="0 0 24 24"`
- Use `stroke="currentColor"` (no hardcoded fills except intentional dots)
- Keep 2px stroke weight from the source library

See `docs/ICONS.md` for the full token table and Figma node IDs.
